## Sobre el proyecto

Web informativa de **Sabores con Eli**, negocio de repostería y comida
para llevar. Sitio estático (HTML / CSS / JS) desplegado con
**GitHub Pages** y dominio propio a través de **Hostalia**.

Este repositorio forma parte del TFG del ciclo de **Desarrollo de
Aplicaciones Multiplataforma (DAM)**.

## Estructura del proyecto

```
├── index.html          → página principal
├── css/
│   └── styles.css       → estilos (paleta cacao / rosa / dorado)
├── js/
│   └── main.js           → pestañas del menú + animaciones scroll
└── img/
    └── productos/         → imágenes del menú
```

## Tecnologías

- HTML5 · CSS3 · JavaScript (vanilla)
- Tipografías: Fraunces · Work Sans · Space Mono
- Hosting: GitHub Pages
- Dominio: Hostalia (DNS)
